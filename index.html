Quelques rappels sur les tableaux

Soit le tableau suivant : 

    
      let names = [];

Pour boucler dessus on peut faire : 

    
      for (const name of names) {
        console.log('Coucou ' + name);
      }

Et aussi : 

    
      names.forEach(name => console.log('Salut ' + name))

Ici, on appelle la fonction forEach() et l'argument que l'on passe est une fonction !

Et c'est super cool de pouvoir faire ça parce qu'il y a plein de méthodes pratiques qu'on peut "configurer" nous même.

Par exemple on peut trier :

    
      names.sort((name1, name2) => name2.localeCompare(name1));
